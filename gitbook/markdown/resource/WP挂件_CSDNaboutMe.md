## 使用方法：

1、下载后解压

2、把 about-me.css 拷贝到 Wordpress后台的主题目录下，被style.css引用（或者直接拷贝内容进去也可）

3、在前端打开Wordpress的仪表盘， 外观 -> 小工具 -> 自定义HTML， 把 about-me.html 的内容拷贝进去即可

4、about-me.html 内的图片地址修改为自己的图片地址即可
